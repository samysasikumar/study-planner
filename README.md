# 📅 StudyPlan — Personal Schedule
 
My personal weekly study planner and calendar — a clean, minimal tool I built for myself to manage my schedule. Built entirely in HTML, CSS, and vanilla JavaScript with no frameworks or dependencies.
 
🌐 **Live:** [samysasikumar.github.io/study-planner](https://samysasikumar.github.io/study-planner)
 
---
 
## Why I Built This
 
I wanted a simple, distraction-free way to visualize my weekly schedule without signing up for anything or relying on an app. So I built one — it saves automatically and works offline.
 
---
 
## ✨ Features
 
- **Week & Day views** — toggle between a full weekly overview and a single day view
- **Add subjects/events** — name, day, start/end time, color, and repeat pattern
- **Color coding** — 8 color options to visually distinguish subjects
- **Repeat options** — set events to repeat daily, weekly, or just once
- **Edit & delete** — click any event to edit or remove it
- **Today button** — jump back to the current date instantly
- **Subject list** — sidebar showing all added subjects at a glance
- **Persistent storage** — schedule saves automatically via localStorage so it's still there when you come back
- **Keyboard navigation** — arrow keys to move between weeks/days
- **Now line** — live indicator showing the current time on the calendar
---
 
## 🛠️ Built With
 
| Technology | Purpose |
|---|---|
| HTML5 | Structure and layout |
| CSS3 | Styling, grid layout, responsive design |
| Vanilla JavaScript | All app logic, calendar rendering, state management |
| localStorage | Client-side data persistence |
| Inter (Google Fonts) | Typography |
 
> No frameworks, no dependencies — pure HTML/CSS/JS.
 
---
 
## 🚀 How to Run
 
Just open `index.html` in any modern browser — that's it.
 
```bash
git clone https://github.com/samysasikumar/study-planner
cd study-planner
open index.html
```
 
Or visit the live site: [samysasikumar.github.io/study-planner](https://samysasikumar.github.io/study-planner)
